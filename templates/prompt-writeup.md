# Prompt Pattern: Writeup (3-Stage)

> Use case: dev.to / Substack / blog posts
> Used in: 11 dev.to articles + 9 Substack issues

## Stage 1 -- Outline
```
Topic: {{topic}}
Audience: {{audience}}
Angle: {{your_unique_perspective}}

Give me 8 specific sub-points, each with:
- a concrete example or data point
- a sentence hook that makes someone stop scrolling
- why this sub-point matters more than alternatives

Prioritize unexpected counterintuitive points.
```

## Stage 2 -- Draft
```
Expand outline points into full paragraphs.
Rules:
- Start each para with the hook sentence
- No hedging (it seems, maybe)
- Include 1 specific number or example per para
- End each section with a 'so what' sentence
- Target: {{word_count}} words total
```

## Stage 3 -- Polish
```
Read the draft aloud. Flag any sentence where:
- The voice sounds generic (AI-default)
- The logic jumps without evidence
- The paragraph is > 4 sentences (split it)

Rewrite flagged sections.
Then add: 1 counterintuitive opening, 1 memorable closing.
```
